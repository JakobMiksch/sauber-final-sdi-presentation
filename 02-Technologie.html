Verwendete Standards

  • WFS - Web Feature Service
  • WMS - Web Map Service
  • WCS - Web Coverage Service

WMS - Web Map Service

  • Visuelle Darstellung
  • Rasterdaten
  • auch für Vektordaten

WFS - Web Feature Service

  • Vektordaten
  • für Geometrien
  • Punkte, Linine, Polygone, ...

WCS - Web Coverage Service

  • rohe Rasterdaten
  • ohne visuelle Darstellung
